# 🚀 CodeHub

CodeHub is a full-stack developer blogging platform where users can publish posts, explore categories, and interact with technical content.

---

## 🌐 Live Demo

👉 https://thecodeehub.netlify.app/

---

## 🛠 Tech Stack

### 🔹 Backend
- Java  
- Spring Boot  
- JWT Authentication  
- MySQL  
- AWS Elastic Beanstalk  

### 🔹 Frontend
- HTML  
- CSS  
- JavaScript  
- Netlify  

---

## 🔥 Features

- ✅ User Registration & Login  
- ✅ JWT Authentication  
- ✅ Create Post  
- ✅ Delete Post  
- ✅ Categories  
- ✅ View Posts by Category  
- ✅ Profile Page  
- ✅ Secure REST APIs  
- ✅ Backend deployed on AWS  
- ✅ Frontend deployed on Netlify
